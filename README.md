## 🚀 快速开始

### 编译 C 程序
```bash
gcc -o hello hello.c
```

### 编译 C++ 程序
```bash
g++ -std=c++17 -o hello hello.cpp
```

### 使用 CMake 构建
```bash
mkdir build && cd build
cmake ..
cmake --build .
```
